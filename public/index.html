SERVICE

걷기와 환경보호를
한번에 해결하는 솔루션

🎯

스마트 경로 추천

목적지 입력 시 AI가 3가지 경로를 제안
Less Walk, Recommended, More Walk

📱

실시간 내비게이션

GPS 기반 위치 추적과 경로 이탈 감지
안전하고 정확한 도보 안내

📊

건강 데이터 분석

거리, 시간, 걸음 수 자동 측정
개인 건강 기록 관리

FEATURES

핵심 기능

3가지 맞춤 경로

Less Walk: 최단 거리로 빠르게

Recommended: 적당한 운동량

More Walk: 건강을 위한 긴 경로

각 경로별 예상 거리, 시간, 걸음 수를 미리 확인할 수 있습니다.

실시간 GPS 네비게이션

정확한 GPS 추적으로 현재 위치와 진행률을 실시간 표시

경로에서 40m 이상 벗어나면 자동으로 재탐색 안내

목적지까지 남은 거리와 시간을 지속적으로 업데이트

건강 기록 관리

매일의 걷기 데이터를 자동으로 저장하고 분석

총 거리, 소요 시간, 걸음 수를 한눈에 확인

지속적인 건강 관리와 목표 설정을 지원
